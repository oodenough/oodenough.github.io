# Debian Server Usage

## some log files

```bash
/var/log/syslog 	messages from the kernel, system daemons, and applications
/var/log/auth.log 	authentication-related messages, such as user logins and  						authentication failures
/var/log/messages 	general system messages that do not fit into other log files
/var/log/apache2/
/var/log/mysql/
```
