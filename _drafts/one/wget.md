-O <filename>: Save the downloaded file under a different name.
-P <directory>: Save the downloaded file in a specific directory.
-c: Resume a partially downloaded file.
-q: Quiet mode. Suppress output.
-r: Recursive download. Downloads files from subdirectories as well.
-np: Do not ascend to the parent directory while recursively downloading.

wget -O newfile.zip https://example.com/file.zip -P /path/to/directory
