```bash
sudo apt-get install python3-pip #==> install pip(python的包管理器)
pip install pygame #==> install pygame module using pip
```
write a python "hello world"
```bash
print("hello world") #==> save as name.py
```
then execute it with python3 `python3 name.py`

**打包为二进制文件**
```bash
pip install pyinstaller
./.local/bin/pyinstaller hello.py
```
打包后二进制文件位于`dist/hello/hellp`
