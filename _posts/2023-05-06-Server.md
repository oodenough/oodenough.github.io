Login as `root` first :-)

## sshd => enable ssh login (Arch)
1. set up your system 
2. enable internet connection
3. add your `rsa pub key` as `~/.ssh/authorized_keys`
4. start `sshd`
5. connect => `ssh root@ip.add.re.ss`

## httpd => start apache web server (Arch)
1. add a `index.html` in `/srv/http/` directory
2. start `httpd` with `systemctl start httpd`
3. browse to your server's ip

## vsftpd => ftp (very secure ftp daemon)
1. `/etc/vsftpd.conf`
2. `/usr/share/doc/vsftpd`
