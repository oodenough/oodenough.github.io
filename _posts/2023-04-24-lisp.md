`gcl` stands for GNU Common Lisp
```bash
sudo apt-get install gcl
```
gcl is like a interpreter 
type `gcl` to start the interpreter
type `(quit)` to exit the interpreter
type `(load "hello.lisp")` to interprete lisp source code from hello.lisp

* update 23/04/26
there are `levels` in Lisp. `Load` your function on level1 and then you can `call` it on level2
