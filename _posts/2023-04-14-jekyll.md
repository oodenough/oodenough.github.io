## jekyll installation

[jekyll-docs](https://jekyllrb.com/docs/) 

###### install jekyll:
```bash
sudo apt-get install ruby-full build-essential zlib1g-dev
gem install jekyll bundler #==> Install the jekyll and bundler gems
jekyll new blogsite #==> generate a new jekyll site
cd blogsite #==> change directory to the site
bundle exec jekyll serve #==> Build the site and make it available on your pc
```
then open a browser and browse to `127.0.0.1:4000`

###### add your information to your site
edit the `_config.yml`file to add you personal information
add post in `_posts`directory and name the post like `YYYY-MM-DD-name.md`
then open a browser and browse to `127.0.0.1:4000`

#### upload to github pages(user/org pages)
create a public repo named `username.github.io` 
```bash
cd blogsite #==> change directory to the blogsite
git init
git add .
git commit -m "Initial commit"
git branch -M main 
git remote add origin https://github.com/username/username.github.io.git
git push -u origin main
```
then browse to `https://username.github.io`
