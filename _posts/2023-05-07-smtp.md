## 简介
What is `smtp`**Simple Mail Transfer Protocol** ?
[英文维基](https://en.wikipedia.org/wiki/Simple_Mail_Transfer_Protocol) [中文维基](https://zh.wikipedia.org/zh-cn/%E7%AE%80%E5%8D%95%E9%82%AE%E4%BB%B6%E4%BC%A0%E8%BE%93%E5%8D%8F%E8%AE%AE)

我的用途：通过邮件把服务器信息发送到我的电子邮箱

## 安装 (Archlinux)
从`aur`仓库安装
```bash
git clone https://aur.archlinux.org/ssmtp.git
cd ssmtp.git
makepkg -si
```

## Configuration

配置文件在`/etc/ssmtp/`目录下

1. 修改`ssmtp.conf`文件
* 备份
```bash
cp ssmtp.conf ssmtp.conf_bak
```
* modify (QQmail)
执行 `vim ssmtp.conf` 修改如下：
```bash
mailhub=smtp.qq.com:465 #==> 587也可以
RewriteDomain=qq.com
Hostname=yourhostname
FromLineOverride=YES
UseTLS=YES

AuthUser=emailaccount@qq.com
AuthPass=kdjfkssjgj #==> QQ邮箱申请smtp验证密码
```

2. 添加`revaliases`文件
Create aliases for local usernames 

执行`vim /etc/ssmtp/revaliases` 内容如下：
```bash
root:username@qq.com:smtp.qq.com:465
mainuser:username@qq.com:smtp.qq.com:465
#==> root 跟 mainuser都是用户名
```

3. Change the `From` text 
=> by editing /etc/passwd to receive mail from 'root at myhost' instead of just 'root'.
```bash
chfn -f 'root at myhost' root
chfn -f 'mainuser at myhost' mainuser
```
检查是否修改成功：
```bash
$ grep myhost /etc/passwd
root:x:0:0:root at myhost,,,:/root:/bin/bash
mainuser:x:1000:1000:mainuser at myhost,,,:/home/mainuser:/bin/bash
```

## 发邮件

* send email from the terminal 
```bash
$ echo -e "Subject: this is the subject\n\nthis is the body" | mail emailaccount@qq.com
```
* or send interactively
```bash
$ sendmail emailaccount@qq.com
Subject: this is my subject
CC: optional@qq.com

Now I can type the body here
```
`note`==> When using mail interactively, after typing the Subject: subject and other headers, hit enter twice, and then type the body. Hit Ctrl+d on a blank line to end your message and automatically send it out.

* 编辑好文件后发送
```bash
$ vim test-mail.txt

To:username@example.com
From:youraccount@gmail.com
Subject: Test

This is a test mail.
Send the test-mail.txt file

$ sendmail -t < test-mail.txt
```
