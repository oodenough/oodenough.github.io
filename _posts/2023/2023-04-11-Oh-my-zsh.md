## 简介


![oh-my-zsh](/postimgs/2023/oh-my-zsh.png)


[oh-my-zsh](https://ohmyz.sh/) 是 `zsh` 的一个框架(framework)，在该框架下可以为 `shell` 安装插件和主题，既能提高效率又能美化终端。

本文简单介绍如何安装使用。

## 安装 zsh

```bash
sudo apt-get install zsh
chsh -s $(which zsh)
```

## 安装 oh-my-zsh

```bash
sh -c "$(curl -fsSL https://raw.github.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

## 插件 

* syntax-highlighting plug

```bash
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting
```

* auto-suggestion plug 

```bash
git clone https://github.com/zsh-users/zsh-autosuggestions.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions
```

* 配置文件 ~/.zshrc

```bash
plugins={git vi-mode zsh-syntax-highlighting zsh-autosuggestions} 
```

## 主题 

* 安装 powerlevel10k

```bash
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/themes/powerlevel10k
```

* 配置文件 ~/.zshrc

```bash
ZSH_THEME="powerlevel10k/powerlevel10k" #=> meslo fonts needed
```

* 主题设置

`p10k configure` #=> 重新设置主题
