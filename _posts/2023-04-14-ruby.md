## ruby installation

[ruby](https://www.ruby-lang.org/en/)

[installation_docs](https://www.ruby-lang.org/en/documentation/installation/)

###### install ruby:
```bash
sudo apt-get install ruby-full #==> install ruby
ruby -v #==> check the ruby version
gem update --system #==>to upgrade to the latest RubyGems
#==> RubyGems is a package management framework for Ruby
```

###### write a ruby hello-world

```bash
puts "hello world" #==> save as hello.rb
ruby hello.rb 
```
