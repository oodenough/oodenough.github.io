```bash
sudo apt-get install git #==> install git
git -v #==> check the version
git config --global user.name "Nancy" #==> add config info
git config --global user.email "yoyoyo@gmail.com" #==> add config info
```

we can use git to clone a repo from github
```bash
git clone https://github.com/torvalds/linux #==> clone the linux source code use https URL
```

then we can connect our pc to github with SSH
```bash
ssh-keygen -t rsa -C "add some of your info here" #==> this generate a ssh key pair
#==> default path is ~/.ssh/
ls ~/.ssh #==> list the key files 
```
there are `id_rsa` and `id_rsa.pub` in .ssh directory
keep `id_rsa` safe and add `id_rsa.pub` to your github account

then we clone our own repo
```bash
git clone git@github.com:username/reponame.git #==> clone your own repo using SSH URL
```
this repo is cloned with SSH and you have the only `id_rsa` ssh key paired with the `id_rsa.pub` added to your github account
so > you can execute `git push` command with this repo
