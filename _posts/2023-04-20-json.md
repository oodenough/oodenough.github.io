JSON stands for `JavaScript Object Notation`. We can use this `data format` to **exchange data between web applications and servers**.

* JavaScript Object
```js
const cat = {
    name : "Nana",
    age : 3,
    meow : function () {
        console.log(this.name + " is " + this.age " years old");
    }
}
```
JSON looks like a `stringified` version of a regular JS object.
We can create jsonVariable like a regular object {} but wrapped with single quotes '.
A JSON object contains a collection of `key-value pairs`, similar to a JS object.
However, in a JSON, we need to include the keys between quotes
* JSON Variable
```json
const jsonString = '{
    "story": "Life",
    "times": 2,
    "fiction": true
}'
```
JSON objects can gather static data like Numbers, Arrays, Booleans, or Strings. However, they cannot store functions inside.

We can convert a JS object to a JSON with the `JSON.stringify()` method.
```js
const cat = {
    name : "Nana",
    age : 3
}

console.log(JSON.stringify(cat));

#==> below is output
{"name":"Nana","age":3}
```

We can also transform a JSON string into a JS object with the JSON.parse() method.
```js
const jsonString = '{
    "story": "Life",
    "times": 2,
    "fiction": true
}' #==> this won't work in a .js file

#==> key-value pairs have to be in one line
const jsonCat = '{"name": "Nana", "age": 3}'
console.log(JSON.parse(jsonCat));
#==> below is output
{ name: 'Nana', age: 3 }
```
